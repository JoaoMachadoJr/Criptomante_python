Sobre o Projeto

Olá!

Eu me chamo João Machado. Sou aluno do curso de Ciência da Computação da Universidade Federal Fluminense, campus de Rio das Ostras.

Em 2020 irei fazer a entrega de meu Trabalho de Conclusão de Curso (TCC) e esse site faz parte de meu TCC.

Meu orientador nesse projeto é o Dr. Carlos Bazílio Martins.

O tema de minha monografia se entitula "Criação de estimativas para o valor da criptomoeda Bitcoin utilizando comentários extraídos de uma rede social"

Nesse projeto criei uma aplicação capaz de criar estimativas para o valor futuro do Bitcoin baseado no seu histórico de preço e em comentários obtidos de conversas de usuários da rede social Reddit.

Para isso, utilizei técnicas relacionadas a Mineração de Dados, Inteligência Artificial, Aprendizado de Máquina e Processamento de Linguagem Natural.

Nesse site, na sessão APRESENTAÇÃO disponibilizei uma explicação geral sobre como cada técnica foi aplicada para alcançar o objetivo do projeto.

Nesse projeto, desenvolvi 4 formas de análise como meios de estimar o valor futuro do Bitcoin. Onde uma dessas análises, chamada de Análise Consolidada, é um agregamento das outras três.

Todas as previsões e análises são baseadas na cotação do Bitcoin em dólar americano.

Todas as análises se baseiam no princípio de que todo comportamento humano, em perspectiva coletiva, possui tendências cíclicas.

Esse princípio é extensivamente explorado pelo Princípio de Ondas de Elliott, descrito por  RalphNelson Elliott.

A parte técnica do projeto inclui:

  • Programação orientada a objetos, utilizando Python
  • Banco de dados, utilizando PostgreSql 12
  • Automatização de leitura de páginas WEB e extração de informação (Web Crawling)
  • Extração de informação de arquivos CSV
  • Consumo de APIs REST
  • Multithreading
  • Processamentode linguagem natural (inglês), utilizando Spacy
  • Aprendizado de Máquina, utilizando Scikit-Learn
  • Produção de um website, utilizando Django, Html5, Css3 e javaScript

O código-fonte completo do projeto está disponível no github.
